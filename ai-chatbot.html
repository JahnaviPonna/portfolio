🤖 AI Chatbot

An intelligent conversational assistant built with Gemini 1.5 Pro API and Flask. This chatbot provides natural, human-like interactions, ideal for learning how to connect frontend interfaces with large language models (LLMs).

🔧 What It Does

  • Takes user input in a chatbox format
  • Sends queries to Gemini API via Flask backend
  • Receives real-time intelligent responses
  • Displays replies in a chat bubble interface

🖼️ How It Works – Screenshots

1️⃣ Chat UI Input

User types a message in the chatbot interface.

2️⃣ Gemini Response

The chatbot responds based on Gemini API’s output.

3️⃣ Interactive Session

Continues conversation in a fluid, human-like way.

🧰 Tech Stack

  • Frontend: HTML, CSS, Tailwind CSS, JavaScript
  • Backend: Python (Flask)
  • API: Gemini-1.5-Pro API (Google Generative AI)
  • Hosting: Localhost / Render

🎯 Key Highlights

  • Real-time conversation using Gemini API
  • Responsive, mobile-friendly UI
  • Custom error handling and empty input detection
  • Clear display of both user and bot messages
